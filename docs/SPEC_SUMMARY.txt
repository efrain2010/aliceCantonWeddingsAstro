================================================================================
LAYOUT PATTERNS
================================================================================

Hero Section:
  - Full-width, full viewport height (900px+)
  - Background image (cover, parallax effect)
  - Centered circular logo overlay

Content Sections:
  - Full-width container
  - Internal max-width ~1200px, centered with margin auto
  - Padding: 60-80px top/bottom
  - Column layouts: 2-col (50/50), 3-col equal width, or 4-col grid
  - Gap between items: 30-40px

Images:
  - object-fit: cover (fill aspect ratio, crop as needed)
  - Wedding photos often overlaid with dark gradient/opacity for text readability
  - Gallery appears in masonry or justified grid

Buttons:
  - Border-radius: subtle (4-8px)
  - Padding: 15-20px vertical, 40-50px horizontal
  - Text: all-caps
  - Font-weight: 400-600

Forms:
  - Labels in gray text (#A4A4A4)
  - Input fields: light gray background (#F0F0F0), border on focus
  - Placeholder text: lighter gray
  - Required fields marked with red *

================================================================================
KEY DESIGN FEATURES
================================================================================

1. Sticky Header
   - Navigation remains fixed at top during scroll

2. Circular "A" Logo (Hero Overlay)
   - Concentric rings with "WEDDINGS & EVENTS" curved text
   - Large gold script "A" in center
   - SVG or CSS circles (SVG recommended for scalability)

3. Parallax Hero
   - Background image moves slower than scroll (subtle effect)
   - CSS: background-attachment: fixed OR JS library

4. Blurred Background Images
   - Testimonial section uses garden/floral bokeh background
   - Text cards have dark semi-transparent overlay (rgba(0,0,0,0.6))
   - Creates visual depth and text contrast

5. Gallery/Masonry
   - Testimonial gallery shows wedding portfolio in varied image sizes
   - CSS Grid or Columns for layout (no JS required in Astro)

6. Typography Treatment
   - All headings in all-caps Cinzel serif (elegant, formal)
   - Script logo for personal/artisanal touch
   - Generous line-height (30px) for readability
   - Warm gold accents for luxury feel

7. Whitespace
   - Generous spacing between sections
   - Breathing room around content
   - Elegant, uncluttered aesthetic

================================================================================
CRITICAL NOTES FOR ASTRO IMPLEMENTATION
================================================================================

Technology:
  - Reference is WordPress; Target is Astro (static site)
  - Form submission requires external service (Netlify Forms, Formspree, etc.)

Language:
  - Reference is Spanish-only
  - Target is bilingual (Spanish root `/`, English `/en/`)
  - All content must be translated

Typography:
  - Reference uses Cinzel + Halant fonts
  - Target project has Playfair Display + Lato tokens
  - DECISION: Match reference fonts or use project tokens?

Colors:
  - Reference heading color (#6C6975) is different from project charcoal (#2C2C2C)
  - Reference gold (#E5AA45) is warmer than project gold (#C9A96E)
  - DECISION: Use reference colors for fidelity or adapt to project tokens?

Images:
  - All images in reference are high-quality wedding photography
  - Must source or create equivalent images for Astro version
  - Ensure proper licensing

Components (per Astro project structure):
  - Page wrapper: src/pages/index.astro (ES), src/pages/en/index.astro (EN)
  - Page content: src/components/pages/HomePage.astro
  - Shared UI: Header, Footer, ContactForm (use getLang() for i18n)
  - Translations: Inline in HomePage.astro + shared strings in src/i18n/translations.ts
